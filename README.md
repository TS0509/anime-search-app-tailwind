# Anime Search App (Enhanced Tailwind Edition)

## How to run

```
npm install
npm run dev
```

App runs on [http://localhost:4000](http://localhost:4000)

## Tech Stack
- React 18 + TypeScript
- Redux Toolkit
- React Router DOM
- TailwindCSS
- Axios + AbortController
